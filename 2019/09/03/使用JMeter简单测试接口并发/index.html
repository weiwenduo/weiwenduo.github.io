1. 准备工作

测试工具：Apache JMeter 5.1.1

2. 操作流程

2.1. 创建线程组

  • 线程数：同时请求的数量
  • Ramp-Up时间：线程之间间隔的时间
  • 循环次数：该线程组循环运行的次数

2.2. 添加HTTP请求

2.3. 在『线程组』中添加『HTTP信息头管理器』

2.4. 添加『用表格查看结果』

2.5. 点击『启动』即可看到执行结果

3. 查看测试结果

测试结果表格中中，各参数的意义如下：

  • Sample：每个请求的序号
  • Start Time：每个请求开始时间
  • Thread Name：每个线程的名称
  • Label：Http请求名称
  • Sample Time：每个请求所花时间，单位毫秒
  • Status：请求状态，如果为勾则表示成功，如果为叉表示失败。
  • Bytes：请求的字节数

在下面还有几个参数：

  • 样本数目：也就是上面所说的请求个数，成功的情况下等于你设定的并发数目乘以循环次数
  • 平均：每个线程请求的平均时间
  • 最新样本：表示服务器响应最后一个请求的时间
  • 偏离：服务器响应时间变化、离散程度测量值的大小。
